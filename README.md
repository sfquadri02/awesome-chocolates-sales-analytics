# Awesome Chocolates Sales Analytics Dashboard

## Project Overview
This project focuses on analyzing chocolate sales data using Power BI Desktop to identify
revenue trends, product performance, and regional sales insights that support
business decision-making.

## Business Questions Addressed
- Which chocolate products generate the highest revenue?
- How does sales performance vary across regions?
- What are the top and bottom performing products?
- Are there any observable sales trends over time?

## Tools and Technologies
- Power BI Desktop
- Microsoft Excel
- DAX
- Data Modeling
- Exploratory Data Analysis (EDA)

## Key Insights
- Identified top-performing products based on total revenue
- Observed variations in sales performance across regions
- Analyzed sales trends to highlight business opportunities
